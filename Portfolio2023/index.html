skills

HTML

Hypertext Markup Language is used to creat web documents.It defines the syntax and placement of special instructions (tags) that aren't displayed,but tell the browser how to display the document's content.It is also uded to create links to other documents,either locally or over a network such as internet.

CSS

Cascading style sheet Language is used to describe the presentation of an HTML document, and how elements should be rendered on the screen.

JAVASCRIPT

Javascript is a programming Language that allows the implementation of features on web Pages.JavaScript is the scripting language of the Web. JavaScript is used in millions of Web pages to add functionality, validate forms, detect browsers, and much more.

SQL

Structured Query Language, abbreviated as SQL, is a language that is largely used in the industry to query data from databases.

REACT

React creates a virtual DOM. When state changes in a component it firstly runs a "diffing" algorithm, which identifies what has changed in the virtual DOM. The second step is reconciliation, where it updates the DOM with the results of diff.
